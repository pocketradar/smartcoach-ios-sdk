# ``SmartCoachSDK``

Welcome to the **SmartCoach iOS SDK** documentation.

This SDK provides APIs for:

- Discovering SmartCoach radar devices via Bluetooth Low Energy (BLE)
- Connecting to a selected device
- Receiving real-time speed measurements
- Managing configuration and entitlements
- Handling errors through a unified error model

This documentation includes articles, a tutorial, and API reference material
to help you integrate SmartCoach speed measurement into your application.
